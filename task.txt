Тестове завдання на позицію Junior PHP Developer at Softoria, Dataforseo project

Мета
Показати позицію заданого сайту в органічній видачі Google за вказаним ключем, локацією та мовою, використовуючи DataForSEO API v3.

Завдання
1. Ознайомитися з документацією https://docs.dataforseo.com/v3
2. Зареєструватися в дашборді: https://app.dataforseo.com/?aff=37683
3. Використати розділ  SERP -> Google -> Organic.
4. Розробити односторінковий застосунок на Laravel, який міститиме:
   a. Форму з полями: пошукове слово, назва сайту, локація, мова.
   b. Кнопку **«Пошук»**, що надсилає запит до API.
   c. Область результатів, де відображатиметься ранг сайту в органічній видачі (або повідомлення про те, що сайт не знайдено).
5. Завантажити застосунок на GitHub з інструкцією, як його запустити локально.

Технічні вимоги
* Застосунок має запускатися локально та коректно виконувати завдання, описане вище.
